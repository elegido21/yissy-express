Activa la sincronización para ver tus datos en todos tus dispositivos (celular, laptop, tablet). Tus datos se cifran con tu contraseña antes de salir del dispositivo — ni los servidores pueden leerlos.

🤝 Opción A: Sesión P2P Recomendado · Fácil

Crea una sesión con un nombre y contraseña. Comparte ese nombre+contraseña con quien quieras que acceda a tus datos. Cualquier dispositivo que entre a la misma sesión verá los mismos datos en tiempo real.

No requiere configurar Firebase ni crear proyectos. Funciona con tecnología P2P (Gun.js) usando relays públicos gratuitos.

Crear o unirse a una sesión

¿Cómo funciona?

  1. Tú creas la sesión con un nombre (ej: YissyExpress2024) y una contraseña (ej: miClave99)
  2. Compartes el nombre y la contraseña con quien quieras (por WhatsApp, en persona, etc.)
  3. Esa persona abre la app en su dispositivo → entra a esta sección → escribe el mismo nombre y contraseña → click "Crear / Unirse"
  4. ¡Listo! Ambos dispositivos tendrán los mismos datos y se sincronizarán en tiempo real

⚠️ Importante: La contraseña de la sesión NO es la misma que tu PIN de entrada. La sesión se usa para sincronizar entre dispositivos. El PIN se usa para entrar a la app.

🔒 Tus datos se cifran con la contraseña de sesión antes de salir de tu dispositivo. Si olvidas la contraseña de sesión, no podrás acceder a los datos sincronizados (pero tendrás copia local en cada dispositivo).

🔥 Opción B: Firebase Cloud Avanzado

Para usuarios que quieren su propio proyecto Firebase con persistencia garantizada. Más confiable para uso intensivo pero requiere configuración.

Mostrar configuración Firebase
📖 Guía paso a paso Firebase

Paso 1: Entra a https://console.firebase.google.com e inicia sesión con tu cuenta Google.

Paso 2: Click en "Agregar proyecto" → ponle un nombre (ej: yissy-express) → Continuar → Crear proyecto.

Paso 3: En el panel del proyecto, busca el icono Web </>. Click ahí.

Paso 4: Pon un apodo (ej: YISSY Web) → NO marques Firebase Hosting → Registrar app.

Paso 5: Te muestra firebaseConfig con 6 campos. Cópialos y pégalos arriba.

Paso 6: Menú lateral → Build → Firestore Database → Crear base de datos → "Iniciar en modo de prueba" → Habilitar.

Paso 7: En Firestore → pestaña Reglas, pega esto y publica:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Paso 8: Completa el campo "ID de documento" con una frase secreta única.

Paso 9: Click Guardar Firebase y luego Probar conexión.

Copia de seguridad
